{% trans "The Foundation of Great Email Design" %}

{% trans "Well-designed emails get opened, read, and clicked. This guide covers proven principles for creating emails that perform." %}

{% trans "üí° Pro Tip:" %} 60% of emails are opened on mobile devices. Always design mobile-first.

{% trans "Layout and Structure" %}

  • {% trans "Single column layout:" %} Works best on all devices
  • {% trans "Clear hierarchy:" %} Use headings to guide readers
  • {% trans "Logical flow:" %} Most important content first
  • {% trans "White space:" %} Don't cram too much content
  • {% trans "Scannable:" %} Use bullets, short paragraphs

{% trans "Color and Branding" %}

{% trans "Consistent branding builds recognition and trust:" %}

  • Use your brand colors (2-3 primary colors)
  • {% trans "Ensure sufficient contrast for readability" %}
  • {% trans "Avoid bright backgrounds that hurt eyes" %}
  • {% trans "Use color strategically to highlight CTAs" %}
  • Keep background simple (usually white or light gray)

{% trans "Typography" %}

{% trans "Font Choices" %}

{% trans "Stick to web-safe fonts: Arial, Georgia, Helvetica, Times New Roman, Verdana" %}

{% trans "Font Sizes" %}

{% trans "Body text: 14-16px | Headlines: 22-28px | Minimum: 14px for readability" %}

{% trans "Line Height" %}

{% trans "Use 1.5-1.7 line height for comfortable reading" %}

{% trans "Images" %}

{% trans "Use images wisely:" %}

  • {% trans "Optimize size:" %} Keep under 1MB per image
  • {% trans "Alt text:" %} Always include descriptive alt text
  • {% trans "Relevant:" %} Images should support your message
  • {% trans "Responsive:" %} Images should resize on mobile
  • {% trans "Format:" %} Use JPG for photos, PNG for graphics
  • {% trans "Balance:" %} Don't make emails image-only (spam filters)

{% trans "‚ö†Ô∏è Important:" %} Many email clients block images by default. Your email should make sense without images.

Call-to-Action (CTA) Design

{% trans "Your CTA is the most important element:" %}

  • {% trans "One primary CTA:" %} Don't confuse readers with multiple options
  • {% trans "Button style:" %} Use button-styled links, not plain text
  • {% trans "Contrasting color:" %} Make it stand out
  • {% trans "Action-oriented text:" %} "Get Started", "Download Now", "Shop Sale"
  • {% trans "Size:" %} Large enough to tap on mobile (44x44px minimum)
  • {% trans "Placement:" %} Above the fold and repeated if email is long

{% trans "Mobile Optimization" %}

{% trans "Essential mobile design principles:" %}

  1. {% trans "Use single-column layouts" %}
  2. {% trans "Make text at least 14px" %}
  3. Use large, tappable buttons (minimum 44x44px)
  4. {% trans "Keep subject lines under 40 characters" %}
  5. {% trans "Front-load important content" %}
  6. {% trans "Test on actual mobile devices" %}

{% trans "Content Guidelines" %}

  • {% trans "Concise:" %} Get to the point quickly
  • {% trans "Scannable:" %} Use headings, bullets, short paragraphs
  • {% trans "Personal:" %} Use "you" and personalization tags
  • {% trans "Value-focused:" %} Lead with benefits, not features
  • {% trans "Clear:" %} Avoid jargon and complexity
  • Urgent (when appropriate): {% trans "Create FOMO with limited offers" %}

{% trans "Accessibility" %}

{% trans "Make emails accessible to everyone:" %}

  • Use sufficient color contrast (4.5:1 ratio)
  • {% trans "Add alt text to all images" %}
  • {% trans "Use semantic HTML headings" %}
  • {% trans "Avoid all-caps text" %}
  • Make links descriptive (not "click here")
  • {% trans "Test with screen readers when possible" %}

{% trans "Common Mistakes to Avoid" %}

  • ‚ùå Too much text (keep it under 200 words)
  • {% trans "‚ùå Multiple CTAs competing for attention" %}
  • {% trans "‚ùå Images without alt text" %}
  • ‚ùå Tiny fonts (under 14px)
  • {% trans "‚ùå Too many different fonts or colors" %}
  • {% trans "‚ùå No mobile testing before sending" %}
  • {% trans "‚ùå Ignoring email client differences" %}

{% trans "Testing Your Designs" %}

  1. {% trans "Send test emails to yourself" %}
  2. Check on multiple devices (phone, tablet, desktop)
  3. Test in different email clients (Gmail, Outlook, Apple Mail)
  4. {% trans "Verify all links work" %}
  5. {% trans "Check image display and alt text" %}
  6. {% trans "Read through for typos and errors" %}

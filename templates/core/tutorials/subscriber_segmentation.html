{% trans "What is Segmentation?" %}

{% trans "Segmentation divides your subscriber list into smaller, targeted groups based on shared characteristics. This allows you to send more relevant, personalized campaigns." %}

{% trans "üí° Pro Tip:" %} Segmented campaigns can have 2-3x higher open rates and click-through rates compared to non-segmented campaigns.

{% trans "Types of Segmentation" %}

{% trans "Demographic" %}

{% trans "Age, gender, location, occupation, income" %}

{% trans "Behavioral" %}

{% trans "Purchase history, email engagement, website activity" %}

{% trans "Engagement Level" %}

{% trans "Highly active, moderately active, inactive, cold" %}

{% trans "Customer Journey Stage" %}

{% trans "Awareness, consideration, purchase, retention, advocacy" %}

{% trans "Segmentation Filters" %}

{% trans "Available filter types:" %}

  • {% trans "Subscriber info:" %} Email, name, signup date, location
  • {% trans "Campaign activity:" %} Opened, clicked, did not open
  • {% trans "List membership:" %} Is/is not on specific lists
  • {% trans "Tags:" %} Has/doesn't have specific tags
  • {% trans "Custom fields:" %} Any custom data you've collected
  • {% trans "Engagement score:" %} Overall activity level

{% trans "Advanced Segmentation" %}

{% trans "Combine multiple conditions:" %}

{% trans "Example 1: Engaged Local Customers" %}

  • {% trans "Location is 'New York'" %}
  • {% trans "AND Opened any campaign in last 30 days" %}
  • {% trans "AND Has tag 'Customer'" %}

{% trans "Example 2: Re-engagement Targets" %}

  • {% trans "Has not opened any email in 90 days" %}
  • {% trans "AND Was active before that" %}
  • {% trans "AND Is not tagged 'Unengaged'" %}

{% trans "Dynamic vs. Static Segments" %}

{% trans "Dynamic Segments" %}

{% trans "Automatically update as subscribers meet/don't meet criteria. Best for ongoing campaigns." %}

{% trans "Static Segments" %}

{% trans "Fixed list of subscribers at a point in time. Best for one-time campaigns." %}

{% trans "Common Segment Ideas" %}

  • {% trans "VIP customers:" %} High purchase value + frequent buyer
  • {% trans "New subscribers:" %} Joined in last 30 days
  • {% trans "At-risk customers:" %} No purchase in 6 months but was active
  • {% trans "Product interested:" %} Clicked Product X link
  • {% trans "Cart abandoners:" %} Started checkout but didn't complete
  • {% trans "Winback targets:" %} Inactive for 90+ days
